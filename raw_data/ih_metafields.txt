IH Metadata descriptions:

ih data contains snapshots of archive.org for ih locations page, but is much more incomplete than wh

status: string, Current/Closed
state: string, full name
city: string, full name only first letter capitalized
address: string, full street address
note: string, field returns empty, probably need to bash this one around a bit
loc_raw: numeric, loc used for geospatial tagging
full_address: string, combining for tagging
latitude: numeric, has some NAs due to tidy tagging mishaps
longitude: numeric, has some NAs due to tidy tagging mishaps
