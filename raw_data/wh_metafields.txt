WH Metadata descriptions:

wh data contains snapshots of archive.org for waho locations.

store_code: int, internal reference of unique id for stores
business_name: string, full name from pull
city: string, city in caps
state: string, state code
latitude: numeric standard
longitude: numeric standard
status_code: flag; string, A = Operating Normally, CT = Closed Temporarily
business_hours_json: string, json of standard operating hours
formatted_business_hours: string, created text field for my analysis to identify non 24 hour wahos
special_hours_json: string, json of temporary change in business hours
custom_json: string, returned json from notes field, seems to denote franchisee owner
closed_reason: string, returned in the data pull but only returns NA, likely defunct
snapshot_date: datetime, snapshot from archive
state_fips: string, state signifier
cz_fips: string, county signifier
county: string, named county
